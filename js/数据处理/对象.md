## 遍历
```js
// 遍历 keys（不含继承的）
// Object.keys() 
var obj = { foo: 'bar', baz: 42 };
Object.keys(obj) // ["foo", "baz"]
// 遍历 values（不含继承的）
Object.values(obj)
// 遍历 entries（不含继承的）
Object.entries(obj) // [["foo", "baz"], [ "baz", 42 ]]
```

## 冻结
```js
// 冻结
Object.freeze(obj)
// 作为参数传递的对象与返回的对象都被冻结
// 所以不必保存返回的对象（因为两个对象全等）
var o = Object.freeze(obj);
o === obj; // true
Object.isFrozen(obj); // === true
```
